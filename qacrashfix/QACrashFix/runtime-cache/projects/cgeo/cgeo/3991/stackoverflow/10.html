Issue

Your problem consists here: you are trying to create another instance of you activity, inside your fragment:

private void switchFragment(Fragment fragment) {
if (getActivity() == null) {
    return;
}

MainActivity mainActivity = new MainActivity();
    mainActivity.switchContent(fragment);
}

Solution

Instead of this you should getActivity inside fragment and write this code in SlidingMenuFragment :

private void switchFragment(Fragment fragment) {
    if (getActivity() == null) {
        return;
    }

    MainActivity mainActivity = (MainActivity)getActivity();
    mainActivity.switchContent(fragment);
}

It would be a good practice if you would use listeners to communicate to your activity from your fragment
